# BallonGo

## 注意
このプロジェクトはAIを多用していてバグも存在します

## 概要
Trees Hate You　のような死にゲーを参考に制作

## 使用技術
- Unity 2022.3.62f2
- C#
- DOTween

## 担当
個人制作

## 工夫した点
・Factoryパターン
・Spawner/Initializer/Runner構成
・GC削減

## 制作背景
ひとつ前の産学連携プロジェクトで選考に落ち、精神的に不安定な状態になってしまったので
今度こそ面白いゲームを作ってやろうと思い立ち制作

## 目標
2ブロック上の注意と同様の内容ですが 短期間で遊べるレベルまで上げるため
AIを多用し、バグがあっても気にせず友人には口頭で説明します

## 結果
合計11人に遊んで貰い"こんな所から飛び出してくるのか"、"これに当たってもダメなの"などとワイワイしながら 遊んで貰えて、私の心はとてもホクホク！　ゲーム制作への意欲も高まりとてもやってよかったプロジェクトでした
